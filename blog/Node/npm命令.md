### 更新包

> npm update <name> -g    更新全局包
>
> npm update <name> --save  更新生产环境包
>
> npm update <name> --save-dev  更新开发环境包



### 查看全局依赖

> npm ls  -g --depth 0

### 查看当前目录下安装的包

> npm list

### 卸载包

> npm uninstall moudleName



### 查看配置信息

>npm config list //查看配置信息
>npm config ls -l //查看所有配置信息

### 配置全局包路径

> npm config set prefix "F:/nodejs"
